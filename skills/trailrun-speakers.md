## 2. Трейлран — `trailrun-speakers.md`

# trailrun-speakers

Навык для поиска спикеров по трейлраннингу и горному бегу для лектория КАНТ. [cite:2]

## Когда использовать этот навык

Используй, когда:
- Пользователь выбирает тему «Трейлран / горный бег» (`/trailrun_speakers`).
- Нужны спикеры по подготовке к трейлам и ультрам, специфике рельефа, экипировке для гор/троп, безопасности в горах/лесу. [web:50]
- Требуется список 3–10 кандидатов с досье и идеями лекций.

## Входные данные (Inputs)

- **Тема лекции**: первый трейл 10–30 км, горный забег, ультра, набор высоты, техника на подъёмах/спусках, палки, питание на дистанции, экипировка.
- **Уровень аудитории**: новички в трейле, опытные шоссейники, переходящие в горы, продвинутые трейлраннеры.
- **География**: приоритет Урал (Екатеринбург, Челябинск, Пермь и др.), далее Россия. [cite:2]
- **Формат**: лекция, разбор профилей трасс, воркшоп по экипу, практическая вылазка.

## Рабочий процесс (Workflow)

1. Уточни бриф: тип трейла, дистанция, рельеф, сезон (лето/межсезонье/зима). [cite:6]
2. Источники:
   - Организаторы трейлов и ультрамарафонов, спикеры на трейл‑ивентах. [web:50]
   - Блогеры и авторы, которые регулярно пишут/снимают про трейл.
   - Сообщества горного бега, клубы, команды.
3. Отбор кандидатов:
   - Опыт участия и/или организации трейлов, способность объяснять особенности трейла любителям.
   - Акцент на безопасность, питание, экипировку и чтение рельефа.
4. Оценка релевантности:
   - Насколько лекция может помочь клиентам КАНТа в выборе снаряжения и подготовке. [cite:2]
5. Список 3–10 кандидатов с досье, идеями форматов.
6. Приоритизация по Уралу, рекомендации по первым контактам. [cite:6]

## Формат ответа (Output Format)

(аналогично другим навыкам — topic, audience_level, region_priority, candidates[], next_steps_recommendations[]).

## Примеры запросов

- «Найди спикеров по трейлраннингу на Урале для лекции про первый горный забег».
- «Кого позвать на лекцию о подготовке к ультратрейлу и выборе экипа».
- «Подбери кандидатов для беседы о безопасности в горах для трейлраннеров».
