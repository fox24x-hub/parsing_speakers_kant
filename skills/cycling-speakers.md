# cycling-speakers

Навык для поиска спикеров по велоспорту (шоссе, MTB, гравий, любительское катание) для лектория КАНТ. [cite:2][web:48]

## Когда использовать этот навык

Используй, когда:
- Пользователь выбирает тему «Велоспорт» (`/cycling_speakers`).
- Нужны спикеры по выбору велосипеда, посадке, тренировкам, безопасности на дороге, обслуживанию, велопоходам/гревел‑поездкам.
- Требуется список 3–10 кандидатов с досье и форматами лекций.

## Входные данные (Inputs)

- **Тема лекции**: первый шоссейник/гибрид/горный вел, фитинг, подготовка к веломарафону, базовое обслуживание, велоинфраструктура, велопоходы.
- **Тип велосипеда/катания**: шоссе, МТБ, гравий, город, туринг.
- **Уровень аудитории**: новички, любители, продвинутые.
- **География**: приоритет Екатеринбург / УрФО. [cite:2]
- **Формат**: лекция, воркшоп по обслуживанию, разбор экипа, совместный выезд.

## Рабочий процесс (Workflow)

1. Уточни бриф: тип велоактивности, дистанции, уровень, формат. [cite:6]
2. Источники:
   - Локальные велоклубы, сервисы, магазины, организаторы велозаездов.
   - Блогеры и YouTube‑каналы по велоспорту и велопоходам.
3. Отбор кандидатов:
   - Экспертиза в выбранном типе катания.
   - Умение объяснять новичкам, опыт реальных поездок/гонок/походов.
4. Оценка релевантности:
   - Темы выбора велосипеда и экипа, обслуживания, безопасного катания.
5. Список 3–10 кандидатов с досье, форматами.
6. Приоритизация по региону и рекомендация контакт‑плана. [cite:6]

## Формат ответа (Output Format)

(как в других skills: topic, audience_level, region_priority, candidates[], next_steps_recommendations[]).

## Примеры запросов

- «Нужны спикеры по выбору первого шоссейного велосипеда для любителей в Екатеринбурге».
- «Подбери кандидатов на лекцию про велопоходы выходного дня и экип».
- «Кого пригласить, чтобы рассказали про обслуживание велосипеда для новичков».
