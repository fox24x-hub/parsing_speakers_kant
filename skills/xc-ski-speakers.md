# xc-ski-speakers

Когда пользователь хочет найти или отфильтровать потенциальных спикеров по теме беговых лыж для лектория спортивного магазина КАНТ (Екатеринбург / УрФО / Россия). [cite:2][cite:6]

## Когда использовать этот навык

Используй этот навык, когда: [page:1]  
- Пользователь выбирает тему «Беговые лыжи» в Telegram‑боте (`/xc_ski_speakers` или через меню зимних видов спорта).  
- Нужно подобрать спикеров под конкретную лекцию/мероприятие, связанную с беговыми лыжами (техника, подготовка к стартам, снаряжение, сервис, тренировки круглый год). [cite:2]  
- Требуется предложить 3–10 кандидатов с кратким досье и идеями форматов лекций для КАНТа. [cite:6]  

Не используй этот навык, если тема не относится к беговым лыжам (в таком случае переключись на другие навыки: `alpine-speakers`, `snowboard-speakers` и т.п.).  

## Входные данные (Inputs)

Запрос пользователя или контекст от бота должен содержать:  
- **Тему лекции по беговым лыжам** (например: техника конькового хода, подготовка к марафону, выбор лыж/ботинок/креплений, сервис и смазка, ОФП летом).  
- **Уровень аудитории**:  
  - новички (первые шаги, “с нуля”),  
  - любители (катаются 1–3 сезона, участвуют в массовых стартах),  
  - продвинутые (регулярно тренируются, бегают марафоны, сильные любители).  
- **География**:  
  - приоритет: Екатеринбург и Свердловская область,  
  - далее: УрФО,  
  - при необходимости: вся Россия (пометить отдельно). [cite:2][cite:6]  
- **Формат**: лекция в магазине, мастер‑класс на улице/стадионе, онлайн‑разбор, Q&A‑сессия.  

Если каких‑то данных нет, сначала задай уточняющие вопросы и только потом переходи к поиску кандидатов. [web:31]  

## Рабочий процесс (Workflow)

1. **Уточни задачу и аудиторию**  
   - Сформулируй для себя короткий бриф: тема, уровень, география, формат, сезон (подготовка к зиме, разгар сезона, межсезонье). [cite:6]  

2. **Определи источники поиска**  
   Приоритетно смотри:  
   - YouTube‑каналы и видео по беговым лыжам (особенно с аудиториями в регионах Урала/Сибири).  
   - Форумы и сообщества по беговым лыжам и лыжным марафонам.  
   - Соцсети (VK, Telegram, иногда Instagram/Рутюб) тренеров, сервисменов, сильных любителей. [cite:1]  

3. **Выдели потенциальных спикеров**  
   Для каждого кандидата проверь:  
   - Регулярно ли он/она публикует контент по беговым лыжам.  
   - Есть ли опыт в статусе тренера, методиста, сервисмена, организатора стартов или сильного любителя. [cite:2]  
   - Понятен ли стиль подачи (доступно для массовой аудитории, а не только “про спорт высших достижений”).  

4. **Оцени релевантность для КАНТа**  
   - Насколько темы кандидата пересекаются с интересами покупателей магазина КАНТ: выбор снаряжения, подготовка к стартам, техника и безопасность.  
   - Есть ли связи с брендами/клубами/соревнованиями, которые могут быть интересны как партнёры. [cite:2]  

5. **Сформируй список кандидатов**  
   - Отбери 3–10 наиболее подходящих людей.  
   - Для каждого составь краткое досье (см. формат ниже).  
   - Предложи 1–2 возможных формата лекций под каждого спикера. [cite:6]  

6. **Приоритизация и следующие шаги**  
   - Отсортируй кандидатов по приоритету: сначала Екатеринбург/УрФО, затем остальные регионы. [cite:2]  
   - Дай рекомендации по следующему шагу: кого связать с КАНТом в первую очередь, где начать с DM/почты, какие аргументы использовать в первом письме. [cite:6]
