# hiking-speakers

Навык для поиска спикеров по походам, треккингу и туризму (пешему, горному, походы выходного дня) для лектория КАНТ. [cite:2][web:51]

## Когда использовать этот навык

Используй, когда:
- Пользователь выбирает тему «Походы / треккинг / туризм» (`/hiking_speakers`).
- Нужны спикеры по планированию маршрутов, выбору экипировки, безопасности, лагерному быту, навигации.
- Требуется 3–10 кандидатов.

## Входные данные (Inputs)

- **Тип активности**: походы выходного дня, многодневные походы, горный треккинг, зимний поход, походы с детьми.
- **Регион**: Урал / ближайшие горные районы / другие регионы России. [cite:2]
- **Уровень аудитории**: новички, любители, более опытные туристы.
- **Фокус**: экипировка, безопасность, маршрут, организация быта, навигация.
- **Формат**: лекция, разбор снаряжения, воркшоп по планированию маршрута, встреча “истории из походов”.

## Рабочий процесс (Workflow)

1. Уточни бриф: сезон, длительность походов, сложность, регион. [cite:6]
2. Источники:
   - Турклубы, гиды, организаторы походов/туров.
   - Блоги и каналы по походам и outdoor‑туризму. [web:51]
3. Отбор кандидатов:
   - Опыт организации походов, знание региона.
   - Акцент на безопасности, экологичности, подборе снаряжения.
4. Оценка релевантности:
   - Темы выбора рюкзаков, палаток, спальников, одежды для походов, навигации, правил безопасности. [web:51]
5. Список кандидатов с досье, форматами лекций.
6. Приоритизация по Уралу и рекомендация по первым контактам. [cite:6]
