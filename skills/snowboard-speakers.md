# snowboard-speakers

Навык для поиска и оценки потенциальных спикеров по сноуборду для лектория спортивного магазина КАНТ (Екатеринбург / УрФО / Россия). [cite:2][cite:6]

## Когда использовать этот навык

Используй этот навык, когда:  
- Пользователь выбирает тему «Сноуборд» (`/snowboard_speakers` или кнопка в зимнем меню бота).  
- Нужно подобрать спикеров под лекции и мастер‑классы: техника катания, фрирайд, парк, безопасность, выбор и настройка оборудования, подготовка к сезону. [cite:2]  
- Требуется список 3–10 кандидатов с кратким досье и идеями форматов лекций. [cite:6]  

## Входные данные (Inputs)

Желательно иметь:  
- **Тему лекции** (техника, фрирайд, парк, детский сноуборд, выбор доски/креплений/ботинок, лавинная безопасность и т.п.).  
- **Уровень аудитории**: новички, любители, продвинутые.  
- **География**: приоритет Екб/Свердловская область → УрФО → вся Россия. [cite:2]  
- **Формат**: лекция в магазине, теория+практика на склоне, safety‑talk, разбор экипировки.  

Если чего‑то не хватает — сначала уточни у пользователя, затем переходи к поиску. [web:31]

## Рабочий процесс (Workflow)

1. Уточни бриф: тема, уровень, регион, формат и сезон (старт сезона, середина, конец, офф‑сезон). [cite:6]  
2. Источники:  
   - YouTube‑каналы по сноуборду (обучение, обзоры, влог‑форматы).  
   - Форумы/сообщества райдеров, группы местных школ и парков.  
   - Соцсети инструкторов, шейперов парков, гидов, сильных любителей. [cite:1]  
3. Отбор кандидатов:  
   - Регулярный контент по сноуборду, понятная подача.  
   - Опыт: инструктор, гид, райдер, организатор кемпов/ивентов.  
   - Фокус на массовую аудиторию и безопасность. [cite:2]  
4. Оценка релевантности для КАНТа:  
   - Есть ли связка с темами магазина: выбор/подбор доски, ботинок, креплений, защита, обслуживание.  
   - Есть ли опыт выступлений/обучения для групп. [cite:2][cite:6]  
5. Сформируй список 3–10 кандидатов, для каждого сделай досье и предложи 1–2 формата лекции.  
6. Отсортируй по региональному приоритету и дай рекомендации, с кем связываться в первую очередь. [cite:6]
