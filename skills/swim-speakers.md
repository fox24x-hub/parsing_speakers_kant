# swim-speakers

Навык для поиска спикеров по плаванию (бассейн, открытая вода) для лектория КАНТ, в том числе в связке с бегом/триатлоном. [cite:2][web:48]

## Когда использовать этот навык

Используй, когда:
- Пользователь выбирает тему «Плавание» (`/swim_speakers`).
- Нужны спикеры по технике стилей, подготовке к стартам в бассейне и на открытой воде, выбору экипировки, безопасности.
- Требуется список 3–10 кандидатов.

## Входные данные (Inputs)

- **Тема лекции**: обучение взрослого с нуля, техника кроля, плавание для бегунов/триатлетов, открытая вода, страх воды, экипировка (очки, костюм, гидра, аксессуары).
- **Тип воды**: бассейн, открытая вода, оба.
- **Уровень аудитории**: новичок, любитель, триатлет‑любитель, продвинутый.
- **География**: приоритет Екатеринбург / УрФО. [cite:2]
- **Формат**: лекция, разбор ошибок по видео, Q&A, dry‑land‑воркшоп.

## Рабочий процесс (Workflow)

1. Уточни бриф: цель (научиться, улучшить технику, старт), тип воды, уровень. [cite:6]
2. Источники:
   - Школы плавания, тренеры, триатлон‑клубы.
   - Блоги и каналы по плаванию и триатлону.
3. Отбор кандидатов:
   - Опыт обучения взрослых, работа с любителями.
   - Упор на технику и безопасность, адекватные нагрузки.
4. Оценка релевантности:
   - Темы экипировки, подготовки к стартам, связки с бегом/велом (триатлон).
5. Список кандидатов с досье и форматами.
6. Приоритизация по региону и рекомендация по контактам. [cite:6]

## Формат ответа (Output Format)

(аналогично другим).

## Примеры запросов

- «Найди спикеров по плаванию для взрослых новичков в Екатеринбурге».
- «Кого позвать на лекцию про подготовку к заплывам на открытой воде».
- «Подбери тренеров, которые работают с бегунами и триатлетами по плаванию».
