# run-speakers

Навык для поиска и оценки потенциальных спикеров по бегу (шоссейный, городской, любительский) для лектория спортивного магазина КАНТ. [cite:2][cite:6]

## Когда использовать этот навык

Используй, когда:
- Пользователь выбирает тему «Бег» (`/run_speakers` или через меню летних видов спорта).
- Нужны спикеры по подготовке к забегам 5–42 км, базовой технике бега, планированию тренировок, выбору обуви и экипировки, профилактике травм.
- Требуется список 3–10 кандидатов с кратким досье и идеями форматов лекций.

## Входные данные (Inputs)

Желательно иметь:
- **Тема лекции**: первый 5/10 км, подготовка к полумарафону/марафону, бег для здоровья, техника, пульсовые зоны, питание и восстановление, выбор кроссовок и экипа.
- **Уровень аудитории**: новички, любители, продвинутые.
- **География**: приоритет Екатеринбург / УрФО, при необходимости — вся Россия. [cite:2]
- **Формат**: лекция, разбор планов тренировок, Q&A, практическая встреча/совместный лёгкий бег.

## Рабочий процесс (Workflow)

1. Сформулируй бриф: тема, уровень, регион, формат, целевой старт (если есть). [cite:6]
2. Источники:
   - Беговые блоги и YouTube‑каналы (особенно региональные).
   - Локальные беговые клубы и тренеры.
   - Соцсети организаторов забегов, амбассадоров брендов, сильных любителей. [web:47][web:50]
3. Отбор кандидатов:
   - Регулярный контент по бегу, понятная подача для любителей.
   - Опыт: тренер, лидер бегового клуба, организатор забегов или сильный любитель с понятной историей прогресса.
4. Оценка релевантности для КАНТа:
   - Темы выбора обуви/экипировки, подготовки к массовым стартам, безопасных нагрузок.
5. Список 3–10 кандидатов с досье и форматами лекций.
6. Приоритизация по региону и рекомендации, кого позвать первым. [cite:6]
