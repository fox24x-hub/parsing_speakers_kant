# alpine-speakers

Навык для поиска и оценки потенциальных спикеров по горным лыжам для лектория КАНТ. [cite:2][cite:6]

## Когда использовать этот навык

Используй, когда:  
- Пользователь выбирает тему «Горные лыжи» (`/alpine_speakers` или через зимнее меню).  
- Нужны спикеры по технике, подготовке к стартам, выбору/подбору и обслуживанию горнолыжного снаряжения, безопасности на склоне. [cite:2]  

## Входные данные (Inputs)

- **Тема лекции**: техника (параллельные лыжи, карвинг), подготовка к горнолыжному отдыху, оборудование (лыжи/ботинки/крепления/шлем/защита), дети на горных лыжах, безопасность.  
- **Уровень аудитории**: новички, любители, продвинутые.  
- **География**: приоритет Екб/Свердловская область → УрФО → Россия. [cite:2]  
- **Формат**: лекция в магазине, мастер‑класс на склоне, разбор экипировки, Q&A.  

При нехватке данных — уточни вопросы у пользователя. [web:31]

## Рабочий процесс (Workflow)

1. Сформулируй бриф: тема, уровень, регион, формат, сезон. [cite:6]  
2. Источники:  
   - YouTube‑каналы по горным лыжам (обучение, обзоры курортов, снаряжения).  
   - Школы горных лыж, тренеры, клубы, сервисы.  
   - Соцсети инструкторов, тренеров, методистов, сервисменов. [cite:1][cite:2]  
3. Отбор кандидатов:  
   - Регулярный контент по горным лыжам, понятная подача.  
   - Опыт тренерства/обучения/сервиса, участие в ивентах.  
4. Оценка релевантности:  
   - Фокус на любительскую аудиторию и клиента КАНТа.  
   - Умение объяснять про выбор снаряжения и безопасное катание. [cite:2]  
5. Список 3–10 кандидатов с досье, идеями форматов лекций.  
6. Приоритизация по региону и рекомендации по первым контактам. [cite:6]
